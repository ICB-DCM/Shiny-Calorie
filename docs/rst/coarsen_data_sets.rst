coarsen_data_sets
=================

Description
-----------

This function coarsens a data frame according to the numerical
coarsening factor

Usage
-----

.. code:: r

   coarsen_data_sets(df, coarsening_factor)

Arguments
---------

-  ``df``: data frame
-  ``coarsening_factor``: coarsening factor

Examples
--------

.. code:: r

   coarsen_data_sets(values, 2)
