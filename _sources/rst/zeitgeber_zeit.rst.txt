zeitgeber_zeit
==============

Description
-----------

Converts the time to zeitgeber zeit

Usage
-----

.. code:: r

   zeitgeber_zeit(df, light_on)

Arguments
---------

-  ``df``: data frame
-  ``light_on``: indicates when day starts (light on typically)

Examples
--------

.. code:: r

   zeitgeber_zeit(values, 8)
