import_pheno_v8
===============

Description
-----------

This function imports TSE Systems PhenoMaster v8 indirect calorimetry
data sets

Usage
-----

.. code:: r

   import_pheno_v8(file, file_out)

Arguments
---------

-  ``file``: input file
-  ``file_out``: output file

Examples
--------

.. code:: r

   import_pheno_v8(input_file, output_file)
