Code availability and development
=================================

Code is available in the corresponding Github repository: `Shiny-Calorie <https://github.com/stephanmg/calorimetry>`_.

Standalone desktop application installers are available through Sciebo: `Installers <https://uni-bonn.sciebo.de/s/0qDhG2Bu1VNkRli/>`_.

A docker container for the Shiny app is available through Docker.io: `Docker <https://hub.docker.com/r/stephanmg/Shiny-Calorie>`_.
